  <name>task 1: Create PhySO experiment runner</name>
  <files>experiments/physo_experiment.py</files>
  <action>
Create PhySO experiment runner following the gplearn_experiment.py pattern for HPC deployment via jernerics.

1. Create `experiments/physo_experiment.py`:
   - Follow structure of gplearn_experiment.py
   - Import: `from dataclasses import dataclass, field`
   - Import: `from pathlib import Path`
   - Import: `from typing import Any, Dict, Self`
   - Import: `import numpy as np`
   - Import: `from jernerics.experiment import Experiment`
   - Import: `from primel.adapters.physo import PhySOAdapter`
   - Import: `from primel.distributions import Empirical, GaussianKDE, MultivariateUniform`
   - Import: `from primel.samplers import ImportanceSampler, RandomSampler, LHSampler`
   - Import: `from primel.fitness import induced_kl_divergence`
   - Import: `import physo`

2. Define `DATA_DIR = Path(__file__).parent.parent / "data"`

3. Create `@dataclass PhySOExperiment(Experiment)`:
   - Field: `random_state: int`
   - Field: `metrics: dict[str, Any] = field(default_factory=dict)`

4. Implement `setup_data(self, config: Dict[str, Any]) -> np.ndarray`:
   - Load data from `DATA_DIR / config["data_file"]`
   - Return numpy array

 5. Implement `train(self, data: np.ndarray, config: dict) -> None`:
    - Create distributions: Empirical, GaussianKDE (bandwidth from config), MultivariateUniform
    - Create ImportanceSampler with entries: train, kde, uniform
      **IMPORTANT:** Use multi-component sampler matching gplearn_experiment pattern:
      ```python
      sampler = ImportanceSampler(
          sampler_entries=[
              ("train", RandomSampler(empirical_dist), data.shape[0]),  # All training points
              ("kde", RandomSampler(gaussian_kde_dist), config.get("model__n_kde", 100)),  # Local exploration
              ("uniform", LHSampler(uniform_dist), config.get("model__n_uniform", 100)),  # Global exploration
          ],
          random_state=self.random_state,
      )
      # Total samples = data.shape[0] + n_kde + n_uniform
      # PhySO will evaluate expression on ALL these samples, not just training
      ```
    - Create PhySOAdapter with sampler, reference_distribution=gaussian_kde_dist, mean_center_on="train"
   - For each trial (config["n_trials"]):
     - Call `physo.SR()` with:
       - `X = data.T` (PhySO expects n_dim, n_samples)
       - `y = np.zeros(data.shape[0])`
       - `X_names` from config or auto-generate
       - `op_names = ["add", "sub", "mul", "div", "sqrt", "neg", "n2", "inv"]`
       - `run_config = adapter.get_learning_config()`
       - `epochs = config.get("epochs", 100)`
       - `parallel_mode = False`
     - Store metrics (expression, kl_divergence)

6. Implement `_compute_metrics(self, expression, sampler, ref_dist) -> dict`:
   - Evaluate expression on sampler.samples
   - Compute kl_divergence via induced_kl_divergence
   - Return dict with expression string and kl_divergence

7. Implement `evaluate(self, expression, data, config) -> dict`:
   - Aggregate metrics across trials
   - Return summary with num_success, total_kl_divergence

8. Create `get_experiment(config: dict) -> Experiment` factory function

Config defaults (use reasonable values):
```python
{
    "data_file": "zhong/constant.csv",
    "model__kde_bandwidth": 0.1,
    "model__n_kde": 100,
    "model__n_uniform": 100,
    "n_trials": 1,
    "epochs": 10,
}
```

DO NOT:
- Add parallel execution (keep simple like gplearn pattern)
- Add early stopping (PhySO has its own mechanisms)
- Add verbose logging (per user decision)
</action>
